
### │ 소개

가수 백예린의 아티스트 소개 페이지를 디자인 후 반응형으로 제작했습니다

Figma를 사용해 디자인 작업을 진행했고,

Html로 시멘틱한 구조를 잡은 후  javaScript / jquery로 동적인 기능을 추가했습니다
